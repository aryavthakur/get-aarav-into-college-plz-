# CatalystLens

Probabilistic biotech capital-to-catalyst audit backend.

The current codebase lives in `catalystlens_backend/` and exposes a FastAPI
service for solvency modeling, clinical milestone timing, Bayesian technical
success estimates, financing-state valuation, disclosure consistency checks,
and institutional-style Markdown audit reports.

All model outputs are assumption-based estimates for diligence support. They
are not investment advice.
